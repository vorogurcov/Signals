# Задача 1.

Построить порождающий многочлен для кода Рида-Соломона над полем $GF(2^4)$ со следующими характеристиками:
- длина $n=15$
- конструктивное расстояние $d=5$

# Решение

Для кода РС с параметрами $n=15$, $d=5$ порождающий многочлен имеет вид

$$
g(x) = (x - \alpha^b)(x - \alpha^{b+1})(x - \alpha^{b+2})(x - \alpha^{b+3}),
$$

где $b=1$. Так как в поле характеристики 2 $x - \alpha^i = x + \alpha^i$, то

$$
g(x) = (x + \alpha)(x + \alpha^2)(x + \alpha^3)(x + \alpha^4).
$$

Произведения:

1.

$$
   (x + \alpha)(x + \alpha^2) = x^2 + (\alpha + \alpha^2)x + \alpha^3.
$$

   Из таблицы поля $GF(2^4)$: $\alpha + \alpha^2 = \alpha^5$.  

   Получаем $A(x) = x^2 + \alpha^5 x + \alpha^3$.

3.

$$
   (x + \alpha^3)(x + \alpha^4) = x^2 + (\alpha^3 + \alpha^4)x + \alpha^7.
$$

   Вычислим $\alpha^3 + \alpha^4$:  

   $\alpha^3 = 1000_2$, $\alpha^4 = \alpha + 1 = 0011_2$, сумма $1000 \oplus 0011 = 1011_2 = \alpha^7$.  

   Получаем $B(x) = x^2 + \alpha^7 x + \alpha^7$.

5. **Перемножаем $A(x)$ и $B(x)$:**

$$
   g(x) = (x^2 + \alpha^5 x + \alpha^3)(x^2 + \alpha^7 x + \alpha^7) =
   x^4 + c_3 x^3 + c_2 x^2 + c_1 x + c_0,
$$

   где

$$
   \begin{aligned}
   c_3 &= \alpha^5 + \alpha^7, \\
   c_2 &= \alpha^7 + \alpha^5\cdot\alpha^7 + \alpha^3 = \alpha^7 + \alpha^{12} + \alpha^3, \\
   c_1 &= \alpha^5\cdot\alpha^7 + \alpha^3\cdot\alpha^7 = \alpha^{12} + \alpha^{10}, \\
   c_0 &= \alpha^3\cdot\alpha^7 = \alpha^{10}.
   \end{aligned}
$$

   Вычисляем каждое слагаемое, используя таблицу поля:
   - $\alpha^5 = 0110_2$, $\alpha^7 = 1011_2$, $\alpha^5 + \alpha^7 = 1101_2 = \alpha^{13}$.
   - $\alpha^{12} = 1111_2$, $\alpha^7 + \alpha^{12} = 1011 \oplus 1111 = 0100_2 = \alpha^2$, затем $\alpha^2 + \alpha^3 = 0100 \oplus 1000 = 1100_2 = \alpha^6$. Итак, $c_2 = \alpha^6$.
   - $\alpha^{12} + \alpha^{10} = 1111 \oplus 0111 = 1000_2 = \alpha^3$.
   - $c_0 = \alpha^{10}$.

Таким образом, порождающий многочлен:

$$
\boxed{g(x) = x^4 + \alpha^{13} x^3 + \alpha^{6} x^2 + \alpha^{3} x + \alpha^{10}}.
$$

# Задача 2.



Рассмотрим РС-код $(15, 9)$ над полем $GF(2^4)$, исправляющий 3 ошибки. Поле задано примитивным многочленом $p(x) = 1 + x + x^4$.

На выходе из канала получено слово:



$$
v(x) = a + a^3 x + a^{10} x^{2} + a^{12} x^{3} + a^{13} x^{4} + a^{3} x^{5} + a^{9} x^{6} + a^{5} x^{7} + x^{8} + a^{10} x^{9} + a^{8} x^{10} + a x^{11} + a x^{12} + a^{13} x^{13} + a^{2} x^{14}
$$

Определите вектор ошибок $e(x)$

# Решение

### Шаг 1. Расчет синдромной последовательности

Для декодирования необходимо вычислить синдромы $S_j = v(\alpha^j)$ для $j \in \{1, \dots, 6\}$. Подставляя значения $\alpha^j$ в полином $v(x)$ и выполняя сложение в поле $GF(16)$, получаем:

$$
\begin{aligned}
S_1 &= 1, \quad &S_2 &= \alpha^{7}, \quad &S_3 &= \alpha^{4}, \\
S_4 &= \alpha^{14}, \quad &S_5 &= \alpha^{2}, \quad &S_6 &= \alpha^{3}.
\end{aligned}
$$

### Шаг 2. Определение коэффициентов локатора ошибок

Составим ключевое уравнение в виде системы линейных уравнений для поиска коэффициентов полинома $\Lambda(x) = 1 + \lambda_1 x + \lambda_2 x^2 + \lambda_3 x^3$:

$$
\begin{pmatrix} 
1 & \alpha^7 & \alpha^4 \\ 
\alpha^7 & \alpha^4 & \alpha^{14} \\ 
\alpha^4 & \alpha^{14} & \alpha^2 
\end{pmatrix} 
\begin{pmatrix} \lambda_3 \\ \lambda_2 \\ \lambda_1 \end{pmatrix} = 
\begin{pmatrix} \alpha^{14} \\ \alpha^2 \\ \alpha^3 \end{pmatrix}
$$

В результате решения системы методом исключения неизвестных находим:
$$ \lambda_1 = \alpha^{2}, \quad \lambda_2 = \alpha^{11}, \quad \lambda_3 = \alpha^{5}. $$
Таким образом, искомый полином: $\Lambda(x) = 1 + \alpha^2 x + \alpha^{11} x^2 + \alpha^5 x^3$.

### Шаг 3. Идентификация локаторов и номеров позиций

Проверкой элементов поля находим корни $\Lambda(x)$, которыми являются $\alpha^{12}, \alpha^8$ и $\alpha^5$. Локаторы ошибок $X_h$ вычисляются как обратные величины к корням:
* $X_1 = (\alpha^{12})^{-1} = \alpha^3 \implies i_1 = 3$
* $X_2 = (\alpha^8)^{-1} = \alpha^7 \implies i_2 = 7$
* $X_3 = (\alpha^5)^{-1} = \alpha^{10} \implies i_3 = 10$

### Шаг 4. Нахождение амплитуд ошибок

Для определения значений ошибок $Y_h$ в найденных позициях воспользуемся уравнениями связи с синдромами:

$$
\begin{cases}
Y_1 \alpha^3 + Y_2 \alpha^7 + Y_3 \alpha^{10} = 1 \\
Y_1 \alpha^6 + Y_2 \alpha^{14} + Y_3 \alpha^{5} = \alpha^7 \\
Y_1 \alpha^9 + Y_2 \alpha^{6} + Y_3 \alpha^{0} = \alpha^4
\end{cases}
$$

Решение данной системы дает следующие значения амплитуд:
```math
$$ Y_1 = \alpha^2, \quad Y_2 = \alpha^5, \quad Y_3 = \alpha^8. $$
```
### Шаг 5. Итоговый вектор ошибок

Объединяя полученные данные о позициях и амплитудах искажений, записываем полином ошибок:

$$ \boxed{e(x) = \alpha^{2} x^{3} + \alpha^{5} x^{7} + \alpha^{8} x^{10}} $$
