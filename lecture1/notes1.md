# Конспект: Теория информации и системы связи

## 1. Модель цифровой системы связи
Упрощённая модель цифровой системы передачи данных.

### Схема прохождения сигнала:
* **Источник данных / информации** (символы).
* **Кодер источника** (сжатие).
* **Кодер канала** (защита от ошибок).
* **Канал с шумом** (двоичный симметричный канал).
* **Декодер канала**.
* **Декодер источника**.
* **Получатель**.

# Схема системы передачи данных

**Источник данных** ➔ *[информ. символы]* ➔ **Кодер источника** ➔ *[кодовые символы]* ➔ **Канал связи** (+ *Шум* из кодера) ➔ **Декодер источника** ➔ **Получатель**

Работа системы ведется с дискретными последовательностями в конечном поле (поле Галуа) из двух элементов:
> $$GF(2) = \{0, 1\}$$

**Пример трансформации данных:**
`[Источник]` --(010110)--> `[Кодер]` --(00110101)--> `[Канал]`

### Характеристики канала:
Для описания канала используется вероятность перехода **P**.
* **$1 - p$** — вероятность безошибочной передачи.
* **$p$** — вероятность инверсии бита (ошибки).

## 2. Блочное кодирование
Процесс преобразования информационных последовательностей в кодовые слова.

| № | Информ. символы ($k$) | Кодовое слово ($n$) |
|:---:|:---:|:---:|
| 0 | 000 | 000000 |
| 1 | 001 | 111111 |
| ... | ... | ... |

**Основные параметры:**
* **Скорость кода**: $R = \frac{k}{n}$, где $k$ — информ. символы, $n$ — длина блока.
* Кодирование ведется в поле **GF(2)** (бинарные данные).

## 3. Теорема Шеннона (1948 г.)
Клод Шеннон определил предел эффективности передачи данных.
Для двоичного симметричного канала в перех. вероятность p  
C = 1 - h(p)
* **Энтропия двоичного ансамбля(h)**: $h(x) = -x \log_2(x) - (1-x) \log_2(1-x)$.
* **Пропускная способность (C)**: Максимальная скорость передачи.
* **Суть**: 
    * Если скорость $R < C$, можно добиться сколь угодно малой вероятности ошибки за счёт увеличения длины кода.
    * Если $R > C$, надежное декодирование невозможно.

## 4. Метрики Хэмминга

### Вес Хэмминга ($w$)
Количество ненулевых элементов (единиц) в векторе.
Если x - код. слово, то $\omega(x)$ - вес Хэмминга.
* Пример: $w(1011) = 3$.

### Расстояние Хэмминга ($d$)
Число позиций, в которых отличаются два вектора.
* Формула: $d(x, y) = w(x \oplus y)$.

## 5. Линейные коды

Предположим, код исправляет ошибки кратности $t$:
$$t \le \left\lfloor \frac{d_{min} - 1}{2} \right\rfloor$$
где $\lfloor x \rfloor$ — наибольшее целое, не превосходящее $x$.

**Линейный код** — код, в котором сумма двух любых кодовых слов тоже является кодовым словом.

Пусть $C$ — множество кодовых слов:
$$\forall x, y \in C : (x + y) \in C$$

Связь расстояния Хэмминга и веса:
$$d(x, y) = w(x + y) = w(z) = w(z + 0) = d(z, 0)$$

Минимальное расстояние для линейного кода:
$$d_{min} = \min_{x, y \in C} d(x, y) = \min_{z \in C} w(z)$$

**Линейный $q$-ичный $(n, k)$ код $C$** — любое $k$-мерное подпространство пространства $F_q^n$ всевозможных векторов длины $n$.

### Порождающая матрица $(n, k)$-кода

* **Матрица размера $k \times n$**, где строки — это базисные векторы.
* **Кодовые слова** — линейная комбинация базисных векторов.

**Обозначения:**
* $G$ — порождающая матрица.
* $m$ — информационное слово.
* $c$ — кодовое слово.

**Формула кодирования:**
$$c = m \cdot G$$

Пусть $h = (h_1, h_2, \dots, h_n)$. Если все кодовые слова удовлетворяют условию ортогональности, то скалярное произведение выглядит так:
$$(\bar{c}_i, \bar{h}) = c_1 h_1 + \dots + c_n h_n$$
