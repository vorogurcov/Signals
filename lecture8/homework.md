# Решение задач по теории сигналов

Решение задач на тему «Конечные поля и регистры сдвига»
---

### 1. Задача 1
**Формулировка:** Для мультипликативной группы поля $GF(5^2)$, считая $x$ примитивным элементом, определить порядки всех элементов поля.

**Дано:** * Поле $GF(q) = GF(5^2)$.  
* Количество элементов $q = 25$.  
* $x$ — примитивный элемент.

**Решение:** 1. **Анализ структуры:** Мультипликативная группа $GF(25) \setminus \{0\}$ состоит из $n = q - 1 = 24$ элементов. Поскольку $x$ — примитивный элемент, группа является циклической, и любой её элемент представим в виде $x^k$, где $k \in \{0, 1, \dots, 23\}$.

2. **Формула порядка:** Порядок элемента $x^k$ в циклической группе порядка $n$ вычисляется как:  
$$\mathrm{ord}(x^k) = \frac{n}{\gcd(k, n)} = \frac{24}{\gcd(k, 24)}$$

3. **Распределение порядков:** Порядок элемента всегда является делителем числа 24. Возможные значения: $\{1, 2, 3, 4, 6, 8, 12, 24\}$.

| Порядок ($d$) | Условие на $k$ ($\gcd(k, 24) = 24/d$) | Количество элементов ($\phi(d)$) | Примеры $k$ |
| :--- | :--- | :--- | :--- |
| **1** | $\gcd(k, 24) = 24$ | 1 | $k=0$ (элемент $1$) |
| **2** | $\gcd(k, 24) = 12$ | 1 | $k=12$ |
| **3** | $\gcd(k, 24) = 8$ | 2 | $k=8, 16$ |
| **4** | $\gcd(k, 24) = 6$ | 2 | $k=6, 18$ |
| **6** | $\gcd(k, 24) = 4$ | 2 | $k=4, 20$ |
| **8** | $\gcd(k, 24) = 3$ | 4 | $k=3, 9, 15, 21$ |
| **12** | $\gcd(k, 24) = 2$ | 4 | $k=2, 10, 14, 22$ |
| **24** | $\gcd(k, 24) = 1$ | 8 | $k=1, 5, 7, 11, 13, 17, 19, 23$ |

### 2. Задача 2
**Формулировка:** Для кодера $(15, 4)$-кода с проверочным полиномом $h(x) = 1 + x + x^4$ описать смену состояний регистра при начальном заполнении $0, 0, 1, 1$. Убедиться, что это код максимальной длины, и определить минимальное расстояние этого кода.

**Дано:** * Параметры кода: $n=15, k=4$.  
* Проверочный полином: $h(x) = 1 + x + x^4$.  
* Начальное состояние: $M_3 M_2 M_1 M_0 = 0011$.

**Решение:** 1. **Логика работы LFSR:** Полином $h(x) = 1 + x + x^4$ определяет точки обратной связи. Новый бит ($M_{new}$), поступающий на вход при сдвиге, вычисляется как сумма по модулю 2 битов в позициях, соответствующих коэффициентам полинома:  
$$M_{new} = M_0 \oplus M_3$$

2. **Таблица состояний:**

| Шаг | Состояние ($M_3 M_2 M_1 M_0$) | Новый бит ($M_0 \oplus M_3$) | Выход ($M_0$) |
| :--- | :--- | :--- | :--- |
| 0 | 0 0 1 1 | $1 \oplus 0 = 1$ | 1 |
| 1 | 1 0 0 1 | $1 \oplus 1 = 0$ | 1 |
| 2 | 0 1 0 0 | $0 \oplus 0 = 0$ | 0 |
| 3 | 0 0 1 0 | $0 \oplus 0 = 0$ | 0 |
| 4 | 0 0 0 1 | $1 \oplus 0 = 1$ | 1 |
| 5 | 1 0 0 0 | $0 \oplus 1 = 1$ | 0 |
| 6 | 1 1 0 0 | $0 \oplus 1 = 1$ | 0 |
| 7 | 1 1 1 0 | $0 \oplus 1 = 1$ | 0 |
| 8 | 1 1 1 1 | $1 \oplus 1 = 0$ | 1 |
| 9 | 0 1 1 1 | $1 \oplus 0 = 1$ | 1 |
| 10 | 1 0 1 1 | $1 \oplus 1 = 0$ | 1 |
| 11 | 0 1 0 1 | $1 \oplus 0 = 1$ | 1 |
| 12 | 1 0 1 0 | $0 \oplus 1 = 1$ | 0 |
| 13 | 1 1 0 1 | $1 \oplus 1 = 0$ | 1 |
| 14 | 0 1 1 0 | $0 \oplus 0 = 0$ | 0 |
| 15 | 0 0 1 1 | (Цикл завершен) | 1 |

Регистр прошел через $2^4 - 1 = 15$ уникальных состояний. Это подтверждает, что полином $h(x)$ примитивный, а последовательность — М-последовательность.  
 **Минимальное расстояние:** Для дуальных кодов Хэмминга (симплекс-кодов) $d_{min}$ вычисляется как:  
$$d_{min} = 2^{m-1} = 2^{4-1} = 8$$
