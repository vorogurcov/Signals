## **Обработка и интерпретация сигналов**
**Тема:** Коды БЧХ (Боуза — Чоудхури — Хоквингема). Циклотомические классы и построение порождающего полинома.

---

## 1. Концепция кодов БЧХ

Коды БЧХ представляют собой расширение идей циклических кодов, позволяющее **целенаправленно проектировать** код с заранее заданным минимальным расстоянием $d$.

**Фундаментальная теорема БЧХ:**
Пусть $g(x)$ — порождающий полином циклического кода над полем $GF(q)$, а $\alpha$ — примитивный элемент расширения $GF(q^m)$. 
Код обладает минимальным расстоянием **не менее** $d$, если его порождающий полином имеет среди своих корней последовательность степеней $\alpha$:
$$g(\alpha^{b+i}) = 0 \quad \text{при } i = 0, 1, \dots, d-2$$
где $b \ge 0$, а $d \ge 2$ — конструктивное расстояние.

**Особенности для двоичных кодов ($q=2$):**
В силу свойств полей характеристики 2, минимальные многочлены четных степеней совпадают с многочленами соответствующих нечетных степеней ($M_i = M_{2i}$). Поэтому для обеспечения исправляющей способности $t$ ($d = 2t+1$) достаточно использовать только **нечетные** степени:
$$g(x) = \text{НОК}\bigl[M_1(x), M_3(x), \dots, M_{2t-1}(x)\bigr]$$

---

## 2. Циклотомические классы

**Определение:**
Циклотомический класс $C_s$ — это множество индексов (степеней), элементы которого имеют общий минимальный многочлен. Для поля $GF(2^m)$ класс формируется путем последовательного возведения в квадрат по модулю $n$:
$$C_s = \{ s, 2s, 4s, 8s, \dots \} \pmod{n}$$
где $n = 2^m - 1$.

**Ключевые свойства:**
* Классы не пересекаются и в совокупности образуют полное множество индексов от $0$ до $n-1$.
* Каждому классу соответствует ровно один уникальный неприводимый **минимальный многочлен**.
* В качестве «имени» (представителя) класса обычно выбирают наименьшее число в наборе.

**Пример для $n=15$ ($m=4$):**
| Класс | Состав элементов | Представитель |
| :--- | :--- | :---: |
| $C_0$ | $\{0\}$ | 0 |
| $C_1$ | $\{1, 2, 4, 8\}$ | 1 |
| $C_3$ | $\{3, 6, 12, 9\}$ | 3 |
| $C_5$ | $\{5, 10\}$ | 5 |
| $C_7$ | $\{7, 14, 13, 11\}$ | 7 |

---

## 3. Минимальные многочлены

**Минимальный многочлен $M_s(x)$** для элемента $\alpha^s$ — это полином наименьшей степени с коэффициентами из базового поля $GF(2)$, для которого $\alpha^s$ является корнем.

**Свойства:**
1.  **Неприводимость:** Его невозможно разложить на множители над $GF(2)$.
2.  **Общность:** Все элементы одного циклотомического класса являются корнями одного и того же $M_s(x)$.
3.  **Делимость:** Любой полином, имеющий корень $\alpha^s$, обязательно делится на $M_s(x)$ без остатка.

---

## 4. Алгоритм синтеза БЧХ-кода

Для построения кода с заданными параметрами необходимо:
1.  Выбрать длину кода $n$ (для примитивных кодов $n = 2^m - 1$).
2.  Сформировать циклотомические классы по модулю $n$.
3.  Определить требуемую исправляющую способность $t$ и конструктивное расстояние $d = 2t+1$.
4.  Выписать последовательность необходимых корней: $\alpha^1, \alpha^2, \dots, \alpha^{d-1}$.
5.  Определить все классы $C_s$, которые содержат эти корни.
6.  Сформировать $g(x)$ как произведение уникальных минимальных многочленов выбранных классов.

**Параметры результата:**
* $n$: длина блока.
* $k = n - \deg(g(x))$: количество информационных бит.
* $d_{min} \ge d$: реальное минимальное расстояние (не меньше заданного).

---

## 5. Практический пример: Код $(15, 7, 5)$

**Задача:** Исправить $t=2$ ошибки $\implies$ конструктивное $d=5$.
1.  Нужные корни: $\alpha^1, \alpha^2, \alpha^3, \alpha^4$.
2.  Классы: $\alpha^1, \alpha^2, \alpha^4 \in C_1$, а $\alpha^3 \in C_3$.
3.  Порождающий полином: $g(x) = M_1(x) \cdot M_3(x)$.
4.  Размерность: $\deg(M_1)=4$, $\deg(M_3)=4$, итого $\deg(g)=8$. $k = 15 - 8 = 7$.

**Сводная таблица для $n=15$:**
| Набор классов | $g(x)$ | $k$ | $d$ | Тип / Название |
| :--- | :--- | :---: | :---: | :--- |
| $C_1$ | $M_1(x)$ | 11 | 3 | Код Хэмминга (15, 11) |
| $C_1 + C_3$ | $M_1 M_3$ | 7 | 5 | БЧХ (15, 7) |
| $C_1 + C_3 + C_5$ | $M_1 M_3 M_5$ | 5 | 7 | БЧХ (15, 5) |
| Все классы | $\prod M_i$ | 1 | 15 | Код повторения |

---

## 6. Классификация и оценки

* **Примитивные коды:** Длина $n$ строго равна $2^m - 1$.
* **Непримитивные коды:** Длина $n$ является делителем $2^m - 1$. 
    > *Пример:* Код Голея $(23, 12, 7)$. Он строится в поле $GF(2^{11})$, так как 23 делит $2^{11}-1=2047$.

**Оценка размерности:**
Для двоичных кодов нижняя граница числа информационных символов определяется как:
$$k \ge n - m \cdot t$$
(каждая новая исправляемая ошибка "забирает" не более $m$ проверочных бит).

---

## 7. Резюме и связи

1.  **Код Хэмминга** — это простейший случай БЧХ-кода с $t=1$.
2.  **Наращивание мощности:** Чтобы увеличить $t$, мы просто добавляем новые минимальные многочлены в произведение $g(x)$, что эквивалентно добавлению условий (строк) в проверочную матрицу $H$.
3.  **Гибкость:** БЧХ позволяет точно балансировать между избыточностью $(n-k)$ и надежностью $(d)$.
